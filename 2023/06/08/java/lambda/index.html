Lambda

发表于2023-06-08|更新于2023-06-08|java

基础介绍

概述

可以对某些匿名内部类的写法进行简化，关注的是对数据进行的操作

核心原则

可推导可省略

基本格式

（参数列表）-> {代码}

省略规则

  • 参数类型可省略
  • 方法体只有一句代码时，大括号，return和唯一一句代码后的分号可省略
  • 方法只有一个参数是，小括号可省略

    方法引用

    方法体只有一个方法的调用（包括构造方法）
    基本格式： 类名或对象名::方法名

Stream流

概述

对集合或数组进行链状流式操作，函数式编程

操作

必须有终结操作

创建流

单列集合：集合对象.stream()

双列集合：转成单列集合后再创建(如map转为Set map.entrySet())
数组：Arrays.stream(数组)或Stream.of(数组)

将stream流转换为并行流

中间操作

  • filter
  • distinct
  • sorted
    sorted()空参时需要实现Comparable，重写compareTo()
    sorted(比较逻辑)
  • limit
  • skip
  • map ：进行计算或者将一个对象转换为另一个对象
  • flatMap ：一个对象转换为多个对象

    终结操作

  • forEach
  • count
  • max
  • min
  • anyMatch
  • allMatch
  • noneMatch
  • findAny
  • findFirst
  • collect
  • reduce

    注意事项

  • 惰性求值（没有终结操作，中间操作不会执行）
  • 流是一次性的（经过一个终结操作后，就不能再被使用）
  • 不会影响原数据

    函数式接口

    概述

    只有一个抽象方法的接口（JDK的函数式接口都加上了@FunctionalInterface）
