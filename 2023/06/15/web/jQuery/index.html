层次选择器

通过Dom元素间的层次关系来获取特定元素,如后代元素,子元素,相邻元素,同辈元素

选择器 描述 返回 示例
$(“ancestor descendant”) 选取ancestor元素里所有的descendant(后代)元素 集合元素 $(“div span”):选取
里所有的元素
$(“parent > child”) 选取parent元素下的child(子)元素 集合元素 $(“div > span”):选取
里元素名为的子元素
$(“prev + next”) 选取紧接在prev元素后的next元素 集合元素 $(“.one + div”):选取class为one的下一个
同辈元素;
等同于$(“.one”).next(“div”)
$(“prev~siblings”) 选取prev元素后的所有siblings元素 集合元素 $(“.one + div”):选取class为one的后面的所有
同辈元素;等同于$(“.one”).nextAll(“div”)

过滤选择器

通过特定的过滤规则来筛选所需的Dom元素

基本过滤选择器

选择器 描述 返回 示例
:first 选取第一个元素 单个元素 $(“div:first”):选取所有
元素的第一个
:last 选取最后一个元素 单个元素 $(“div:last”):选取所有
元素的最后一个
:not(selector) 去除所有与指定选择器匹配的元素 集合元素 $(“div:not(.myclass)”):选取class不是myclass的div元素
:even 选取索引为偶数的所有元素 集合元素 $(“div:even”):选取索引为偶数的div元素
:odd 选取索引为奇数的所有元素 集合元素 $(“div:odd”):选取索引为奇数的div元素
:eq(index) 选取索引为index的元素 单个元素 $(“div:eq(1)”):选取索引为1的div元素
:gt(index) 选取索大于index的元素 集合元素 $(“div:gt(2)”):选取索大于2的div元素
:lt(index) 选取索小于index的元素 集合元素 $(“div:gt(2)”):选取索小于2的div元素
:header 选取所有的标题元素 集合元素 $(“:header”):选取网页中所有的

,

:focus 选取当前获取焦点的元素 集合元素 $(“:focus”):选取当前获取焦点的元素

:contains(text) 选取包含有text的元素 集合元素 $(“div:contains(‘我’)”):选取含有文本’我’的
元素
:empty 选取不包含子元素或文本的空元素 集合元素 $(“div:empty”)
:has(selector) 选取含有选择器匹配元素的元素 集合元素 $(“div:has(p)”):选择含有

元素的

元素

可见性过滤选择器

选择器 描述 返回 示例
:hidden 选取所有不可见的元素 集合元素 $(“:hidden”)
$(“input:hidden”)
:visible 选取所有可见的元素 集合元素

属性过滤选择器

选择器 描述 返回 示例
[attibute] 选取拥有此属性的元素 集合元素 $(“div[id]”):选取拥有属性id的div元素
[attibute=value] 选取属性值为value的元素 集合元素 $(“div[title=test]”)
[attibute^=value] 选取属性值以value开头的元素 集合元素 $(“div[title^=test]”)
[attibute$=value] 选取属性值以value结尾的元素 集合元素 $(“div[title$=test]”)
[attibute*=value] 选取属性值含有value的元素 集合元素 $(“div[title*=test]”)
[attibute|=value] 选取属性值等于value或以value为前缀(value后跟连接字符-)的元素 集合元素 $(“div[title|=en]”)
[attibute1][attibute2] 满足所有条件的元素 集合元素 $(“div[id][title=en]”):选取拥有属性id并且title为en的div元素

子元素过滤选择器

选择器 描述 返回 示例
:nth-child(index) 选取每个父元素下第index个子元素(索引从1开始) 集合元素
:first-child 选取每个父元素下第1个子元素 集合元素 $(“ul li:first-child”)每个ul中的第一个li
:last-child 选取每个父元素下最后1个子元素 集合元素
:only-child 选取每个父元素下唯一1个的子元素 集合元素

:selected 选取所有被选中的元素(下拉列表) 集合元素 $(“select option:selected”)

ajax()

$.ajax(options) 参数以key/value形式存在

参数名称 类型 说明
url String 发送请求地址(默认为当前页面地址)
type String 请求方式(默认为GET)
timeout Number 设置请求超时事件(单位毫秒)
data String或Object 发送到服务器的数据
dataType String 预期服务器返回的数据类型
beforeSend Function 发送请求前可以修改XMLHeepRequest对象的函数
complete Function 请求完成后调用的回调函数(无论请求成功或失败)
success Function 请求成功时调用的回调函数
error Function 请求失败时调用的回调函数
global Boolean 是否触发全局Ajax事件(默认为true)
